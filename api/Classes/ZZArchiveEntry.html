Inherits from NSObject

Overview

The ZZArchiveEntry class represents an entry in the ZZArchive or ZZMutableArchive zip file.

archiveEntryWithFileName:fileMode:lastModified:compressionLevel:dataBlock:streamBlock:dataConsumerBlock:

Creates a new entry.

+ (id)archiveEntryWithFileName:(NSString *)fileName fileMode:(mode_t)fileMode lastModified:(NSDate *)lastModified compressionLevel:(NSInteger)compressionLevel dataBlock:(NSData *( ^ ) ( ))dataBlock streamBlock:(void ( ^ ) ( NSOutputStream *stream ))streamBlock dataConsumerBlock:(void ( ^ ) ( CGDataConsumerRef dataConsumer ))dataConsumerBlock

Parameters

fileName

The file name for the entry. Only ASCII file names are supported.

fileMode

The UNIX file mode for the entry. This includes the file type bits.

lastModified

The last modified date and time for the entry. The time value is only accurate to 2 seconds.

compressionLevel

The compression level for the entry: 0 for stored, -1 for default deflate, 1 – 9 for custom deflate levels.

dataBlock

The callback that returns the entry file data.

streamBlock

The callback that writes the entry file to the stream.

dataConsumerBlock

The callback that writes the entry file to the data consumer.

Return Value

The created entry.

Discussion

The archive entry will choose the first non-nil dataBlock, streamBlock and dataConsumerBlock to supply its data.
